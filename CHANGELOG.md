## [v0.3.0] - 2026-05-31

### Highlights

- Google Sheets support with 5 new tools for reading spreadsheets

### Added

- `skills/sheets`: New Google Sheets skill with 5 tools ([`c2a6511`](https://github.com/plexusone/mcp-google/commit/c2a6511))
- `get_spreadsheet_metadata` tool: Get spreadsheet metadata (title, sheet count, locale, time zone) ([`c2a6511`](https://github.com/plexusone/mcp-google/commit/c2a6511))
- `list_sheets` tool: List all sheets with properties (rows, columns, frozen) ([`c2a6511`](https://github.com/plexusone/mcp-google/commit/c2a6511))
- `get_sheet_values` tool: Get cell values from a specific range with typed output ([`c2a6511`](https://github.com/plexusone/mcp-google/commit/c2a6511))
- `get_sheet_data` tool: Get all data from a specific sheet ([`c2a6511`](https://github.com/plexusone/mcp-google/commit/c2a6511))
- `get_multiple_ranges` tool: Batch get values from multiple ranges ([`c2a6511`](https://github.com/plexusone/mcp-google/commit/c2a6511))
- CLI: `get-spreadsheet-metadata`, `list-sheets`, `get-sheet-values`, `get-sheet-data` commands ([`e2efda4`](https://github.com/plexusone/mcp-google/commit/e2efda4))
- Auth: Add `spreadsheets.readonly` OAuth scope ([`cc0bf01`](https://github.com/plexusone/mcp-google/commit/cc0bf01))

## [v0.2.0] - 2026-05-26

### Highlights

- Composable architecture built on omniskill framework
- Google Docs support with 4 new tools
- Vault-backed credentials via omnitoken

### Added

- `skills/docs`: Google Docs skill with 4 tools ([`62ed848`](https://github.com/plexusone/mcp-google/commit/62ed848))
- `get_document_metadata` tool: Get document metadata (title, word count) ([`62ed848`](https://github.com/plexusone/mcp-google/commit/62ed848))
- `get_document_content` tool: Get structured content with images and tables ([`62ed848`](https://github.com/plexusone/mcp-google/commit/62ed848))
- `get_document_text` tool: Get all text as plain text ([`62ed848`](https://github.com/plexusone/mcp-google/commit/62ed848))
- `get_document_paragraphs` tool: Get text by paragraphs ([`62ed848`](https://github.com/plexusone/mcp-google/commit/62ed848))
